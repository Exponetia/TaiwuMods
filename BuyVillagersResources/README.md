# 平衡的资源共享  

当太吾村庄缺乏资源，而村民却拥有足够的资源时，太吾与村民之间可以建立一种合作关系。  
通过购买/交换村民的物资，实现资源共享。  
双赢才是真的赢 (仁善)，其实就是免去玩家每月跟同道交换资源的时间。  
太吾每月将用溢出的资源，威望，金钱跟村民交易。  
优先级是 : 溢出资源 > 威望 > 金钱。  
由于不想管边缘作用, 当资源超过仓库的百分之八十，我就当作溢出来处理。  
  
购买力大概是 :  
1资源=4金钱  
2资源=1威望。
